## 迁移内容

### 已替换的API调用

1. **loadCustomers()** - 加载客户列表
   - 旧方式：`wx.cloud.callFunction({ name: 'erp-api', data: { action: 'getCustomers' } })`
   - 新方式：`API.getCustomers()`
   - 状态：✅ 完成

2. **loadSuppliers()** - 加载供应商列表
   - 旧方式：`wx.cloud.callFunction({ name: 'erp-api', data: { action: 'getSuppliers' } })`
   - 新方式：`API.getSuppliers({ limit: 1000 })`
   - 状态：✅ 完成

3. **loadData()** - 加载采购订单
   - 旧方式：`wx.cloud.callFunction({ name: 'erp-api', data: { action: 'getPurchaseOrders' } })`
   - 新方式：`API.getPurchaseOrders({ limit: 500 })`
   - 状态：✅ 完成

4. **confirmStockIn()** - 确认入库
   - 旧方式：`wx.cloud.callFunction({ name: 'erp-api', data: { action: 'stockInPurchaseOrder' } })`
   - 新方式：`API.stockInPurchaseOrder(data)`
   - 状态：✅ 完成
   - 特殊处理：入库成功后自动清除缓存

### 代码改进

1. **引入统一日志**：
   - 使用 `logger.error()` 替代 `console.error()`
   - 所有错误都有统一的日志格式

2. **async/await 语法**：
   - 所有函数改为 async 函数
   - 使用 try/catch 统一错误处理

3. **缓存管理**：
   - 入库操作后自动清除采购订单缓存
   - 确保数据实时性
